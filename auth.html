Amazon Login

This helper signs you in with Login with Amazon, then sends the access token back to the LyricVibe desktop app. It uses the direct PKCE flow, so Amazon only needs to allow this page as the return URL.

Ready.

In the Amazon console, allow https://anttothe1.github.io as the JavaScript origin and https://anttothe1.github.io/LyricVibe/auth.html as the return URL.
